在CentOS 6.6 上编译以及安装最新版Git

2016-09-21
Linux
Git

前言

  之前一直没机会接触过Linux系统，恰巧前几天开始接触到CentOS 6.6，当然上来的第一件事，就是安装Git。网上有些教程，可能由于系统的差异性，所以也碰到好一些坑。我希望把这些坑记录下来，避免下次再犯类似的错误或者能迅速解决问题。

工具

  连接远程linux服务器，当然首选Xshell，我自己使用的Xshell5。

配置

  • 连接
    配置：名称、*协议(SSH)、*主机、*端口
  • 用户身份验证
    方法：Public Key
    用户名：Input your user name
    用户密钥：导入保存的SSH Key

连接

  完成上述的配置后就可以连接远程主机。
  管理员登陆：输入命令符-sudo -i

安装Git

  • 安装wget：yum install wget -y
  • 安装依赖包(必须)
    1. yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel
    2. yum install gcc perl-ExtUtils-MakeMaker
  • 获取最近版本：点击查看
  • 路径定向：cd /usr/src
  • 下载源码：wget https://github.com/git/git/archive/v2.10.0.tar.gz
  • 解压文件：tar xzf v2.10.0.tar.gz(解压后 git-2.10.0 文件夹)
  • 编译安装(按照步骤)：
    1. cd git-2.10.0
    2. make prefix=/usr/local/git all
    3. make prefix=/usr/local/git install
    4. echo "export PATH=$PATH:/usr/local/git/bin" >> /etc/bashrc
    5. source /etc/bashrc

  编译安装可能需要一段时间，慢慢等吧！

大功告成

  检查一下版本号：git --version ==> git version 2.10.0
